iMocha Assessment Library — Detailed Catalog

## Overview

iMocha maintains the industry's largest validated skills assessment library with 3,000+ tests across 500+ job roles. All assessments are:
- Built by domain SMEs with 10+ years of industry experience
- Psychometrically validated (content validity, construct validity, criterion validity)
- Reviewed annually for relevance (questions retired if >2 years old in fast-moving domains)
- Available in multiple difficulty tiers: Entry-Level, Mid-Level, Senior, Lead/Architect

## Assessment Configuration Options

### Question Settings
- Time limit: global or per-question; can allow extra time for accessibility accommodations
- Randomization: randomize question order, answer option order, or draw from question pools
- Sectioned assessments: define separate pass/fail thresholds per section
- Adaptive branching: set next question based on previous answer (Bayesian adaptive engine)

### Proctoring Settings (per assessment)
- Video proctoring: ON/OFF
- Screen share recording: ON/OFF
- Browser lockdown: ON (prevents tab switching and external apps)
- AI anomaly detection: flags are scored 0–100 (configurable threshold for auto-flag)
- Human review queue: flagged sessions routed to human reviewer dashboard

### Candidate Experience
- Mobile-friendly assessments (responsive, iOS/Android tested)
- Screen reader compatible (WCAG 2.1 AA)
- Multi-language UI (English, Spanish, French, German, Portuguese, Hindi, Mandarin)
- Custom branding: client logo, color scheme, welcome message, completion redirect URL

## Assessment Quality Metrics

iMocha publishes transparency reports on assessment quality:

- **Item Difficulty Index (p-value)**: ideal range 0.3–0.7; items outside this range reviewed
- **Discrimination Index (D)**: measures whether high-scorers outperform low-scorers on each item; D > 0.25 required
- **Point-Biserial Correlation**: correlation between item score and total score; >0.2 required for retention
- **Test-Retest Reliability**: Cronbach's alpha > 0.80 for all standard assessments
- **Predictive Validity Studies**: iMocha has completed 12 predictive validity studies correlating assessment scores with 12-month performance ratings across 6 client partners; average r = 0.43
