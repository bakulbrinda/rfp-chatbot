iMocha Product Overview — Comprehensive Guide
=============================================

## What is iMocha?

iMocha is an AI-powered skills intelligence platform that helps enterprises assess, validate, and develop talent at scale. Founded in 2015, iMocha serves over 500 enterprise clients across 70+ countries, including Fortune 500 companies, staffing agencies, and global IT firms. The platform combines assessment science, AI proctoring, and skills data to power better hiring, L&D, and workforce planning decisions.

iMocha's core mission: "Make every talent decision data-driven and skills-first."

## Core Platform Modules

### 1. Talent Assessment Platform
iMocha's flagship product enables organizations to build, deploy, and score technical, cognitive, and behavioral assessments at scale.

Key capabilities:
- Library of 3,000+ ready-to-use skills tests (coding, cybersecurity, data science, cloud, ERP, language, soft skills)
- Custom question authoring with AI-assisted item generation
- Adaptive assessment engine that adjusts difficulty in real-time based on candidate performance
- Multi-format questions: MCQ, coding challenges (40+ languages), video responses, simulations, case studies
- AI-powered proctoring: eye-tracking, tab-switch detection, face recognition, audio monitoring, plagiarism detection
- Real-time scoring and instant candidate feedback reports

Supported test categories:
- Programming & coding (Python, Java, JavaScript, C++, SQL, R, etc.)
- Data Science & AI/ML (statistical reasoning, ML model evaluation, data wrangling)
- Cloud & DevOps (AWS, Azure, GCP, Kubernetes, Terraform, Docker)
- Cybersecurity (ethical hacking, SOC analysis, vulnerability assessment, OWASP)
- ERP & Business Applications (SAP, Salesforce, ServiceNow, Oracle, Workday)
- Cognitive abilities (numerical, verbal, logical, critical thinking)
- Behavioral & leadership competencies (DiSC-aligned profiles, situational judgment)
- Language proficiency (English, Spanish, French, German, Mandarin — CEFR aligned)

### 2. Skills Intelligence Cloud (SIC)
iMocha's second-generation product that goes beyond assessment into continuous skills mapping.

- Builds a living Skills Taxonomy for the entire organization — maps employees to skills, proficiency levels, and skill gaps in real-time
- Integrates with HRIS (Workday, SAP SuccessFactors, Oracle HCM) to create a unified employee skills profile
- Skills Gap Analysis: benchmarks current workforce skills against industry standards, competitor profiles, and future role requirements
- Workforce Planning dashboards: visualize bench strength, succession risk, and skills distribution by department, geography, and business unit
- AI Skills Inference: analyzes resume text, LinkedIn profiles, project history, and learning completions to infer skills without requiring a formal test
- Skills Ontology: a proprietary graph of 30,000+ skills with parent-child relationships, synonyms, and cross-domain mappings

### 3. Coding Interview Platform (CodeSpace)
A browser-native collaborative coding environment for live technical interviews.

- Real-time collaborative IDE with 40+ languages and framework support
- Whiteboard mode for system design interviews
- Pair programming mode where interviewers can type alongside candidates
- Pre-built coding problem library (Easy / Medium / Hard, tagged by domain and company)
- AI code analysis: automatically detects brute-force vs optimal solutions, flags time/space complexity
- Interview recording and playback for debrief and compliance

### 4. Talent Analytics
- Predictive Hire Quality Index: correlates assessment scores with 6-month and 12-month performance ratings
- Diversity & Inclusion analytics: tracks demographic breakdown at each funnel stage, flags potential bias in question sets
- Assessment Effectiveness Reports: item-level analytics (difficulty index, discrimination index, point-biserial correlation)
- Benchmark comparison: compare your candidate pool scores against iMocha's global database of 50M+ assessments

## Deployment Modes

iMocha supports three deployment models:

1. **SaaS (Cloud)** — Fully hosted on AWS (primary) with GCP as DR. Multi-tenant architecture with logical data isolation. 99.9% uptime SLA. Data centers in US-East, EU-West (Frankfurt), APAC (Singapore), and India (Mumbai).

2. **Private Cloud / Dedicated Tenant** — Single-tenant deployment in iMocha's cloud for clients requiring data sovereignty. Dedicated compute, storage, and database. Available in additional regions on request.

3. **On-Premises** — Full deployment within client's own data center or VPC. Requires Kubernetes 1.24+ or Docker Compose. iMocha provides a deployment runbook, Helm charts, and 90-day implementation support. Suitable for government and highly regulated industries.

- ISO 27001:2022 certified
- SOC 2 Type II certified (Security, Availability, Confidentiality)

- CCPA compliant
- FedRAMP Ready (In Authorization)

- Secrets management via HashiCorp Vault

### Candidate Data Policies
- Assessment data retained for 12 months by default; configurable to 3 months

- No sale of candidate data to third parties
- Candidate consent flow built into every assessment link
